# Conv Learnings - 2026-05-26

## 1. Path aliases de-risk large file moves
**Topics:** astro, cc-workflow

**Context:** ROUTE-FLIP moved 43 pages + ~30 component files/dirs.

**Learning:** Because every page imports via `@components/`/`@layouts/` aliases (0 relative `../` page imports), moving a file to any depth breaks no imports — aliases resolve absolutely. The flip reduced to pure `git mv` + rewriting only the move *targets* (layout/component alias strings) and `/matt` URL literals. Confirm alias-vs-relative import style *first* on any move-heavy refactor; it determines whether the job is trivial or a per-file import slog.

## 2. Blind URL replace breaks `startsWith` route-matching
**Topics:** astro, cc-workflow

**Context:** §8 said "ControlBar/MainNav: `/matt/` → `/`".

**Learning:** A naive `/matt/`→`/` replace inside a `startsWith('/matt/')` active-state check yields `startsWith('/')` — always true, so every nav item highlights. Home-route matchers need exact-match special-casing (`href === '/'` → `currentPath === '/'`), not blind substitution. Always read path-matching logic before bulk-replacing route prefixes in it.

## 3. Word-boundary hazard in namespace replace
**Topics:** cc-workflow

**Context:** Doc filenames `matt-provenance.md`, `matt-design-system.md` contain "matt".

**Learning:** A `s{/matt\b}` replace would corrupt `/matt-provenance` → `/-provenance` (hyphen is a word boundary). Scope namespace-prefix replaces to the URL shapes only: `s{/matt/}` + `s{/matt(?=['"\`])}` (followed by quote/backtick), never a bare `\b`. Excluded `src/styles` from the bulk pass to avoid mangling `/matt/*` comments into `/*` inside CSS.

## 4. Test suite catches what static analysis can't
**Topics:** testing

**Context:** All static gates (tsc/check/lint/build) passed; the full suite found `onboarding.test.ts` failing.

**Learning:** The test read `readFileSync('src/pages/onboarding.astro')` — a runtime string path tsc can't see. After move-heavy refactors, the full suite is the only gate that catches hardcoded filesystem-path references. Worth the ~3min on structural changes.
