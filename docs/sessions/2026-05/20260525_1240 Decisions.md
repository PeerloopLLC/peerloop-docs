# Conv Decisions - 2026-05-25

## 1. Don't revert the Tailwind-bridge spacing override — migrate legacy onto Matt incrementally
**Type:** Strategic
**Topics:** astro, cc-workflow

**Trigger:** The `tokens-tailwind-bridge.css` `--spacing-*` override (imported globally since Conv 174) shrank every legacy spacing utility ~4× app-wide; the demo home page regressed (AppNavbar 64px not 256px).

**Options Considered:**
1. Full revert of the bridge override + convert legacy to literal-px
2. Migrate legacy ONTO Matt incrementally — retrofit AppNavbar to use what it can from Matt, leaving the override intact ← Chosen
3. Demo-scoped patch / defer

**Decision:** Do not revert. The Matt styling is the final resting place for style and must not be compromised — `/matt` depends on the override. Retrofit legacy components onto Matt tokens/typography incrementally, going slow, using Matt's MainNav as the reference.

**Rationale:** Reverting would break `/matt`, which depends on the override. Incremental migration preserves the target design system while bringing legacy forward.

**Consequences:** NAV-RETROFIT block created. AppNavbar restyled to Matt (220px width, Matt typography, brand-blue flat active pill, chevrons, slideout offsets) as step 1.

**Type:** Implementation
**Topics:** astro

**Consequences:** AppNavbar rows restyled with Matt typography classes and active state; A remains an open question for after B settles.

## 5. Description font = 12px (`text-body-small`)
**Type:** Implementation
**Topics:** astro

**Trigger:** AppNavbar description text was bumped from the original 10px (`text-xxs`) to 14px during restyle, causing wrap.

**Options Considered:**
1. A — 10px (restore original)
2. B — 12px (`text-body-small`) ← Chosen
3. C — 14px (keep)

**Decision:** B — description text set to `text-body-small` (12px), `line-clamp-2` retained.

**Rationale:** 10px was the original but 12px via Matt's `text-body-small` token aligns with the Matt design system while avoiding the 14px wrap.

**Consequences:** AppNavbar description rows render at 12px Matt token with two-line clamp.
