**Type:** Strategic
**Topics:** design-system, tokens, matt-collaboration

**Trigger:** Probe revealed Matt's Note yellow (#FFF6B8 + #F1E9B0 border) is hardcoded hex in Figma, NOT a Variable. Initial draft proposed A) tokens-only, B) full alignment with tokens, C) tokens + track other drifts.

**Options Considered:**

**Decision:** Update Note.tsx to match Matt's exact Figma spec with all values hardcoded inline (background hex, border, dimensions, shadow). No new `--note-yellow` token. The 6 visual drifts (wrong yellow hex, missing border, wrong radius/padding/gap/shadow) all fixed in same edit.

**Consequences:** Note.tsx fully aligned to Figma. No new tokens added. New memory file `feedback_tokenize_only_matt_variables.md` established the principle as standing rule.
