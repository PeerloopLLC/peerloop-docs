## 2. Slot-forwarding fix lives in AppLayout via unconditional Fragment + ternary
**Type:** Strategic
**Topics:** astro, matt-design

**Trigger:** Astro Fragment-forwarding suppressed HeaderBar's slot fallbacks. First-pass fix (`Astro.slots.has + &&` short-circuit in AppLayout) didn't restore the fallback.

**Options Considered:**
1. Keep defaults in HeaderBar's `<slot>` fallback + use `Astro.slots.has + &&` to suppress empty forwarding — DID NOT WORK
2. Move defaults to AppLayout inside ternary in *unconditional* Fragments ← Chosen
3. Always-pass-content from page (no slot defaults at all)

**Decision:** B. Single source of truth for defaults at the layout consumer (AppLayout); HeaderBar becomes a pure shell primitive with no slot fallbacks.

**Rationale:** Collapses the dual-layer fallback into one. Eliminates the slot-forwarding ambiguity entirely. Trade-off: pages using HeaderBar directly (without AppLayout) lose the defaults — acceptable per the component's docstring noting direct use is rare.

**Consequences:** HeaderBar.astro slot fallbacks removed in [MSH-REFINE]; docstring updated to point to AppLayout. Saved learning as `memory/reference_astro_slot_forwarding.md`.

## 3. Tailwind `lg:` breakpoint shifted from 1024px to 1025px globally
**Type:** Strategic
**Topics:** matt-design, tailwind

**Trigger:** [MSH-REFINE] item 3 — Tailwind default `lg:` is 1024px; Matt's spec uses 1025px as the desktop boundary. 1px shift across all `lg:*` callsites.

**Options Considered:**
1. Override `--breakpoint-lg: 1025px` in `tokens-tailwind-bridge.css` — single source of truth, propagates to every `lg:*` (matt AND legacy) ← Chosen
2. Custom `matt:` breakpoint at 1025px — scoped, but verbose
3. Accept the 1px shift — visually identical, only matters at exactly 1024px

**Decision:** A. Global override.

**Rationale:** Matches Matt's spec exactly. Single source of truth. Same low-blast-radius reasoning as the Conv 174 global spacing override.

**Consequences:** Every `lg:*` callsite in matt/* AND legacy fraser/* pages now activates at ≥1025px. Visual impact only at the 1024px edge case.
