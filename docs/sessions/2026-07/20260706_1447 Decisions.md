## 3. Sidebar short/landscape fix — mt-auto + overflow + landscape-gated compaction
**Type:** Implementation
**Topics:** sidebar, css, mobile

**Trigger:** Top/bottom clusters overlap on short viewports; drawer/desktop scroll gaps.

**Options Considered:**
1. A+B structural only
2. A+B + landscape-gated compaction ← Chosen
3. A+B + always-on compaction
4. + smaller font

**Decision:** Replace `justify-between` with `mt-auto`; add `lg:overflow-y-auto` to desktop; compact rows/padding + unify all gaps to 12px + hide WORKSPACES divider/label only under `@media (max-height:500px)`.

**Rationale:** Structural fix is design-neutral and deterministic; compaction gated to short heights keeps Matt spacing in portrait. Uniform seam + WORKSPACES-hide were user refinements (image-driven).

**Consequences:** 3 files; height proxy (≤500px) not exact per-role collision — noted as tunable / JS-collision-detect available if needed.
