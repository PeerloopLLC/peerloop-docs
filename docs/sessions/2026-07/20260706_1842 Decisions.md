# Conv Decisions - 2026-07-06

## 1. SIDEBAR-COLLIDE trigger = JS ResizeObserver + hysteresis (Option A)
**Type:** Implementation
**Topics:** astro, ui-ux

**Trigger:** Conv 367's fixed `@media (max-height:500px)` merge trigger is role-imprecise (a multi-role admin collides at ~540px, above the proxy; a plain student not until ~350px, below it).

**Options Considered:**
1. JS ResizeObserver → `data-merged` on real overflow, with hysteresis ← Chosen
2. CSS row-count buckets (`data-sb-rows` × per-bucket `@media` thresholds) — pure-CSS/VT-safe but still a proxy
3. Tune the single proxy threshold (~540px) + document the residual imprecision

**Decision:** Option A — a ResizeObserver sets `data-merged` only on genuine overflow; the merge presentation (seam + WORKSPACES-hide) keys off it; compaction stays on the `@media` proxy (can't be collision-gated — it shrinks content and would oscillate).

**Rationale:** Only option that's actually precise for the admin>500px band; `transition:persist` helps a JS-attribute approach rather than hurting it (see Learning 4). Verified per-role in-browser (admin ~700px vs student ~650px).

**Consequences:** New (for this island) ResizeObserver measurement pattern; a brief hydration flash possible on very short viewports (SSR renders un-merged). global.css split into a compaction `@media` block + a `[data-merged]` presentation block.

## 2. No staging re-seed for this deploy
**Type:** Implementation
**Topics:** deployment, cloudflare

**Trigger:** User asked to "check if re-seeding is needed" as part of the staging deploy.

**Options Considered:**
1. Deploy only, no re-seed ← Chosen
2. `db:setup:staging:*` (destructive reset + re-seed)

**Decision:** Deploy the Worker only; do not re-seed.

**Rationale:** Changes are frontend-only (no schema/migration/data-shape change), and a read-only query showed staging D1 already healthy (11 users / 6 courses, current schema). A re-seed = destructive `db:reset:staging` wipe for zero benefit.

**Consequences:** `deploy:staging` ran clean; staging data untouched.
