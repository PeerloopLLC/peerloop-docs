## Uncategorized
- **Height-proxy vs true collision:** the SIDEBAR-MOBILE "merged" behaviors (compaction, uniform seam, WORKSPACES-hide) all fire at `max-height:500px`, an approximation of the actual cluster-collision point (which varies by role count — ~540px for admin+creator+teacher, ~350px for a plain student). Works well for the common case; JS collision-detection would make it exact but adds an observer + oscillation risk (compaction shrinks content, which could un-collide). Left as a documented, tunable proxy.
- **Pre-existing Tailwind v4 flag:** `NavDrawer.tsx:77` uses `outline-none` (v4 prefers `outline-hidden`) — surfaced by `check:tailwind` twice this conv, not caused by this conv's edits. Candidate for a dedicated v4-remnant sweep.
