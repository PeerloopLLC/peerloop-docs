## Notable Prompt Patterns

### Comprehensive Page Creation Request
The second prompt is a good template for requesting bulk page creation:
- Specifies what files to update (ROUTES.md, Header, Footer)
- Requests realistic sample content
- Mentions downstream implications (DB schema)
- Uses "ALL" to indicate no cherry-picking

### Marketing Site Audit
The first prompt is useful for auditing any landing/marketing site:
- Focuses on "public-facing" pages
- References existing header/footer links as source of truth
- Frames purpose (explain and sell to potential users)
