# Session Learnings - 2025-12-28

## Technical

### Tailwind v4 Class Renames
- **bg-gradient-to-*** renamed to **bg-linear-to-*** (supports angles like `bg-linear-45`)
- **shadow-sm** → **shadow-xs**, **shadow** → **shadow-sm** (size shift)
- **blur-sm** → **blur-xs**, **backdrop-blur-sm** → **backdrop-blur-xs**
- **rounded-sm** → **rounded-xs**, **rounded** → **rounded-sm**
- **outline-none** behavior changed: now sets `outline-style: none` instead of transparent. Use **outline-hidden** for v3 behavior
- Deprecated: `bg-opacity-*`, `text-opacity-*` → use slash notation (`bg-black/50`)
- Renamed: `flex-shrink-*` → `shrink-*`, `flex-grow-*` → `grow-*`

### Tailwind v4 Tooling Gap
- No official CLI linting tool for Tailwind v4
- eslint-plugin-tailwindcss has only beta v4 support with false positives
- VS Code IntelliSense only analyzes **open files** (not project-wide)
- prettier-plugin-tailwindcss is for sorting only, not validation
- **Solution**: Created `scripts/check-tailwind-v4.sh` to grep for known v3→v4 renames

### Mock Data Consolidation Pattern
- Centralized mock data in `src/lib/mock-data.ts` (~630 lines)
- Export both raw data and helper functions for different views
- Use types like `CourseListItem` for list views, full `Course` for detail views
- Components import only what they need → smaller bundles
- Benefits: single source of truth, easy to add data, ready for API replacement

## Architecture

### Mock Data Helper Functions
```typescript
// List views (browse pages)
getCourseList(): CourseListItem[]
getCreatorList(): CreatorListItem[]

// Featured (homepage)
getFeaturedCourses(): CourseListItem[]
getFeaturedCreators(): CreatorListItem[]

// Detail views
getCourseBySlug(slug): Course & { creator: CreatorSummary }
getCreatorByHandle(handle): Creator
getCreatorWithCourses(handle): Creator & { courses: CourseListItem[] }

// Related
getRelatedCourses(courseId, limit): CourseListItem[]
getCoursesByCreator(creatorId): CourseListItem[]
```

This pattern keeps components clean and makes the future API transition straightforward.
