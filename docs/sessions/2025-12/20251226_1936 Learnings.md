# Session Learnings - 2025-12-26

## Technical

### Astro JSON Imports in Pages Directory
- JSON files in `src/pages/` are treated as page routes by Astro, generating warnings
- Solution: Move JSON data files to `src/data/pages/` and use path aliases (`@data/*`)
- This separation keeps data files from being interpreted as routes

### Reusable Page Spec Component Pattern
- Created `PageSpecView.astro` as a reusable component for rendering page specifications
- All 42 pages share the same component, reducing code duplication significantly
- The component handles all section types: tables, lists, connections, etc.

### Layout Sidepanel Slots
- Added `showSidepanel` prop to both `AppLayout` and `AdminLayout`
- Uses conditional rendering with named slots (`<slot name="sidepanel" />`)
- Pattern matches existing `LandingLayout` implementation for consistency

### Page Generation Script
- Created `scripts/generate-all-pages.ts` to auto-generate all Astro page files
- Uses a layout mapping object to assign correct layout per page code
- Outputs consistent structure: imports, PageSpecView, sidepanel slot content

## Workflow

### Batch Processing Approach
- First parse all markdown specs to JSON (`npm run parse-all-pages`)
- Then generate all Astro pages from JSON (`npm run generate-pages`)
- Two-step process allows for independent updates to either layer

### Cleanup After Refactoring
- When moving from markdown rendering to JSON: remove unused dependencies
- Removed `react-markdown`, `remark-gfm`, `remark-breaks` after switching approach
- Also removed obsolete utility files (`parseOutgoingLinks.ts`, `MarkdownContent.tsx`)

## Architecture

### Three-Layout System
| Layout | Use Case | Pages |
|--------|----------|-------|
| LandingLayout | Public/auth pages | 13 pages |
| AppLayout | Authenticated app pages | 21 pages |
| AdminLayout | Admin SPA pages | 8 pages |

### JSON-Driven Page Rendering
- Page specifications stored as structured JSON in `src/data/pages/`
- JSON includes: metadata, purpose, connections, sections, user stories, API calls, etc.
- Astro pages import JSON and pass to `PageSpecView` component
- Sidepanel shows outgoing navigation links extracted from JSON
