# Session Learnings - 2026-02-16

## 1. Cloudflare Pages Build Image Uses asdf/node-build Internally
**Topics:** cloudflare, deployment

**Context:** Reviewing Cloudflare Preview build output after Session 215 deploy, a warning appeared about Node.js 22 being in "LTS Maintenance mode."

**Learning:** The CF Pages build environment uses `asdf` with `node-build` to install the Node.js version specified in `.nvmrc`. The "LTS Maintenance mode" warning comes from `node-build`, not from Cloudflare itself. It's informational and does not block the build.

---

## 2. Cloudflare Pages Does Not Support Node 24 (Feb 2026)
**Topics:** cloudflare, deployment

**Context:** Investigated whether upgrading from Node 22 to Node 24 was feasible to address the LTS Maintenance warning.

**Learning:** As of February 2026, Node 24 is **not available** in the CF Pages build image. Users who attempt it get `node-build: definition not found: 24`. This is a platform-level blocker — independent of whether Astro or other dependencies support Node 24. The CF build image receives rolling updates, so this will likely be resolved in a future update.

---

## 3. Astro 5.x Officially Supports Node 18/20/22 Only
**Topics:** astro, deployment

**Context:** Checked Astro compatibility as part of evaluating a Node 24 upgrade.

**Learning:** Astro 5.x officially lists Node 18.20.8+, 20.3.0+, and 22.0.0+ as supported versions. Node 24 is not explicitly listed. Astro follows a pattern of supporting even-numbered LTS versions, so Node 24 support is expected but not confirmed. Node 18 will be dropped in Astro 6.0.

---

## 4. Node Version Is Pinned in Two Files for Redundancy
**Topics:** deployment, workflow

**Context:** Audited where the Node.js version is configured in the project.

**Learning:** The project pins Node in `.nvmrc` (read by CF Pages build image and local nvm/asdf) and `package.json` engines (enforced during `npm install`). Both should be updated together. There is no `.node-version` or `.tool-versions` file. Upgrading Node is a two-file change.
