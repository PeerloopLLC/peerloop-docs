# Session Decisions - 2026-02-16

## 1. Stay on Node 22 — Do Not Upgrade to Node 24
**Type:** Implementation
**Topics:** deployment, cloudflare

**Trigger:** Cloudflare Pages build output showed a warning that Node 22 is in "LTS Maintenance mode," prompting investigation of upgrading to Node 24.

**Options Considered:**
1. Upgrade to Node 24 (current Active LTS) — Would silence the warning and get active support
2. Stay on Node 22 until blockers are resolved ← Chosen

**Decision:** Remain on Node 22.19.0. Do not attempt a Node 24 upgrade at this time.

**Rationale:** Two independent blockers prevent upgrading:
1. **Cloudflare Pages** does not include Node 24 in its build image (users report `node-build: definition not found: 24`)
2. **Astro 5.x** does not officially list Node 24 as supported (only 18/20/22)

Node 22 is supported until April 2027, so there is no urgency. The warning is informational only.

**Consequences:**
- No code changes needed
- Updated `tech-011-cloudflare.md` with CF Pages Node version support matrix
- Updated `tech-013-devcomputers.md` with Node.js version section and upgrade path
- Future sessions will see the documented blockers and know not to retry until CF Pages adds Node 24
