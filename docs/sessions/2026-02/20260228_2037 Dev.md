**Claude:** Created `../Peerloop/scripts/route-matrix.mjs` — a Node.js ESM script that:

- Detects layout per page (AppLayout vs AdminLayout) by parsing imports

- Follows component imports 2 levels deep to find links in child components
- Normalizes dynamic segments (`${slug}` → `[slug]`, `/@me` → `/@[handle]`)
- Classifies link types (tab, card, btn, crumb, redir) via URL pattern heuristics
- Applies hardcoded shared nav link sets based on layout
- Outputs PAGE-CONNECTIONS.md and research/ROUTE-ADJACENCY-MATRIX.tsv
