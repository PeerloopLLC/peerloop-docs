## Effective Patterns

1. **Screenshot sharing** - User shared GitHub Actions screenshots to show exact CI errors
2. **Incremental debugging** - Fixed issues one at a time, committed, pushed, checked CI results
3. **Simple commands** - Short prompts like "let's commit" work well for routine tasks
