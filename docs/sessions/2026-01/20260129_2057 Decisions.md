# Session Decisions - 2026-01-29

## 1. Keep Separate canCreateCourses and canTeachCourses Permissions
**Type:** Strategic
**Topics:** auth, d1

**Trigger:** Guy Rymberg (a Creator with 4 courses) had `canTeachCourses=false`, raising the question of whether we need both permissions.

**Options Considered:**
1. Keep both `canCreateCourses` and `canTeachCourses` as separate admin-controlled permissions ← Chosen
2. Remove `canCreateCourses` - derive "is creator" from `createdCourses.length > 0`
3. Merge into single `canTeachCourses` permission

**Decision:** Keep both permissions as separate, admin-controlled flags.

**Rationale:**
- Admin may need to revoke a user's ability to create courses independently of teaching
- Admin may need to revoke a user's ability to teach independently of creating
- Granular control supports moderation and quality control scenarios

**Consequences:**
- Updated Guy and Gabriel Rymberg seed data to have `canTeachCourses=1`
- Both staging and production databases re-migrated

## 2. Creator Auto-Added as ST with Self-Service Control
**Type:** Strategic
**Topics:** auth, workflow

**Trigger:** Clarifying the relationship between Creators and Student-Teachers for their own courses.

**Options Considered:**
1. Creator must manually add themselves as ST
2. Creator is auto-added as ST, can remove themselves later ← Chosen
3. Creator is always ST of their course (cannot opt out)

**Decision:** When a Creator creates a course, they are automatically added as one of its Student-Teachers. The Creator can then add or remove themselves as an ST of their own course (self-service).

**Rationale:**
- Most Creators will want to teach their own course initially
- Allows Creators to step back and let other STs handle teaching
- Self-service reduces admin burden for common case

**Consequences:**
- Creators need `canTeachCourses=1` to be eligible for auto-ST assignment
- Business logic for course creation should create ST record for creator
- UI should allow Creator to manage their own ST status per course
