## Key Insights

1. **Tailwind v4 uses CSS native syntax** - `bg-linear-to-b` is correct, not `bg-gradient-to-b`

2. **Skills should be project-level** - Global skills (`~/.claude/commands/`) aren't version controlled

3. **Zod provides single source of truth** - Schema defines both types and validation, reducing maintenance burden

4. **Runtime validation catches JSON errors** - TypeScript doesn't validate JSON imports at compile time
