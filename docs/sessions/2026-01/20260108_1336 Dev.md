## Technical Notes

### Database Query Patterns for Admin

**Users:**
```sql
COUNT(*),
SUM(CASE WHEN suspended_at IS NULL AND deleted_at IS NULL THEN 1 ELSE 0 END) as active,
SUM(CASE WHEN suspended_at IS NOT NULL THEN 1 ELSE 0 END) as suspended,
SUM(CASE WHEN created_at >= datetime('now', '-7 days') THEN 1 ELSE 0 END) as new_this_week
```

**Courses:**
```sql
SUM(CASE WHEN is_active = 1 AND suspended_at IS NULL THEN 1 ELSE 0 END) as published,
SUM(CASE WHEN badge = 'featured' THEN 1 ELSE 0 END) as featured
```

**Enrollments:**
```sql
SUM(CASE WHEN status IN ('enrolled', 'in_progress') THEN 1 ELSE 0 END) as active,
SUM(CASE WHEN status = 'completed' THEN 1 ELSE 0 END) as completed
```

**Revenue:**
```sql
COALESCE(SUM(amount_cents), 0) as total_revenue
FROM transactions WHERE status = 'completed'
```
