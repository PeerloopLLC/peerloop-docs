# Development Session Log - 2026-01-09

## Work Completed

### 1. Fixed Vitest `it.runIf()` Timing Issue
- **Problem:** Tests using `it.runIf(!machine?.supportsWorkerd)` were running on all machines because `machine` was undefined at test definition time
- **Root Cause:** `machine` was set in `beforeAll()` but `it.runIf()` evaluates conditions at module parse time
- **Solution:** Moved machine detection to module scope (top-level `const machine = detectMachine()`)
- **Commit:** `032210b`

### 2. Skip `.dev.vars` Test on CI
- **Problem:** Test checking for `.dev.vars` failed on CI because that file is gitignored (contains secrets)
- **Solution:** Added `isLocalDev` check - only run on known local machines (MacMini, MBA-2017), skip on "Unknown" (CI)
- **Commit:** `92c250e`

### 3. Added Wrangler as Dev Dependency
- **Problem:** E2E tests failed with `wrangler: not found` on CI
- **Root Cause:** Wrangler was only installed globally on dev machines, not in package.json
- **Solution:** `npm install --save-dev wrangler`
- **Commit:** `8afaa9b`
