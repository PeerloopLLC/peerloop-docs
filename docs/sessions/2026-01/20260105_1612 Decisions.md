# Session Decisions - 2026-01-05

## Decision 1: External Video Embeds for MVP

**Type**: Strategic
**Driver**: User-driven
**Working On**: Block 3 - Course Content planning
**Trigger**: Planning phase - need to decide how video content would be delivered

**Options Considered**:
1. **External embeds (YouTube/Vimeo)** <- **Chosen**
   - Use existing video_url field in curriculum table
   - Embed via iframe with platform detection
   - Simple, no storage costs, familiar UX
2. **R2 video hosting**
   - Upload videos to R2 bucket
   - Stream via Workers
   - Full control but complex transcoding, higher cost

**Decision**: Use external video embeds (YouTube/Vimeo) for Block 3 MVP.

**Rationale**:
- Simpler implementation for Genesis cohort
- No R2 streaming infrastructure needed
- Creators likely already have videos on YouTube/Vimeo
- Can add R2 streaming later if needed

**Consequences**:
- Created `VideoEmbed.tsx` with YouTube/Vimeo detection
- R2 used only for downloadable resources, not video
- R2 video streaming deferred to Post-MVP Phase 15
