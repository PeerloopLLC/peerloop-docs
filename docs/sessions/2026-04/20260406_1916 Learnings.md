## 3. HS512 JWT Generation Works in Bash with OpenSSL
**Topics:** bbb, security, bash

**Context:** Needed to generate JWT Bearer tokens for bbb-analytics trigger (different auth from URL-embedded HMAC)

**Learning:** `openssl dgst -sha512 -hmac` produces valid HS512 signatures. Combined with base64url encoding of header/payload, a complete JWT can be built in pure bash without external tools. Tested and verified against Web Crypto.

**Pattern:** `echo -n "${header}.${payload}" | openssl dgst -sha512 -hmac "$SECRET" -binary | openssl base64 -e | tr -d '\n=' | tr '+/' '-_'`
