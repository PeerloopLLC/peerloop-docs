### CB2 — Calendar back-nav stuck at May

### New Patterns
- **Pre-computed `downloadUrl` in SSR loaders** — loader decides file-vs-link semantics; component renders `href={resource.downloadUrl}` blindly. Applied to `CommunityDetailResource`.
- **Three-tier auth short-circuit** (creator → admin → member) in `community-resources/[id]/download.ts`. Each tier fires a single-row query only if the previous tier returned false.
- **Resource schema parity** between community_resources and session_resources — r2_key/external_url/size_bytes/mime_type columns, identical type CHECK enum. Makes cross-table helpers and shared upload UI possible later.
