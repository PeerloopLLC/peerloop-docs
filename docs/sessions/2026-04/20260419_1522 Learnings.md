# Conv Learnings - 2026-04-19

## 1. Skill Name Field vs Directory Name
**Topics:** cc-workflow, skills-system

**Context:** User asked to remove "@" from skill names for `/r-end`, `/r-commit`, and `/r-timecard-day`. The actual issue was a `2` suffix in the `name:` frontmatter field of each SKILL.md file (e.g., `r-end2`), not the directory name.

**Learning:** The skill system registers skills by the `name` field in SKILL.md frontmatter, not by the directory name. The directory (`r-end/`) and the registered name (`r-end2`) can diverge silently. When a user refers to "the skill name," clarify whether they mean the directory name or the `name:` frontmatter field — these can differ and the divergence is not immediately obvious from the skill list.
