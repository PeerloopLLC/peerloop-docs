## 1. Temporary Post-Build Patch for CF Pages Deploy
**Type:** Implementation
**Topics:** cloudflare, deployment, astro

**Trigger:** CF Pages build failed; need staging deploy for client review today

**Options Considered:**
1. Migrate Pages → Workers now (permanent, but 1-2 hours with unknowns)
2. Pin adapter to 12.x (likely incompatible with Astro 6) ← Rejected
3. Post-build script to patch wrangler.json ← Chosen for today

**Decision:** Ship a temporary `postbuild` script (`scripts/fix-pages-wrangler.mjs`) on the staging branch to remove invalid fields from the generated wrangler.json. Plan Workers migration as a proper PLAN.md block.

**Rationale:** Fastest path to a client-visible staging deploy. Script is 5 lines of logic, easily removable.

**Consequences:** Staging deploys work today. CF-WORKERS block added to PLAN.md for permanent fix. Script is fragile — will break on adapter updates.
