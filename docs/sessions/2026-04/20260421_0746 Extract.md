### New Patterns
- **Partial unique index + INSERT OR IGNORE** for "at most one open X per Y" invariants that must coexist with legitimate state transitions (close → reopen)
- **COALESCE backfill in UPDATE** for "set if null, preserve if set" semantics without read-modify-write
- **Strict narrowing cascade** of cron detection passes: each pass sees a smaller candidate set than the last; completed items drop out naturally
