# Conv Learnings - 2026-04-07

## 1. TypeScript 6.0 is the Final JS-Based Release
**Topics:** typescript, tooling

**Context:** Researching whether to upgrade TS during package review

**Learning:** TypeScript 6.0 (released 2026-03-23) is explicitly a bridge release — the last JavaScript-based compiler before the Go rewrite in TS 7.0. The ecosystem must pass through it. Breaking changes are modest (compilerOptions.types defaults, rootDir behavior). Astro 6 already supports it via a merged commit from February 2026.

## 2. Astro 5→6 Requires Coordinated Adapter Upgrades
**Topics:** astro, cloudflare

**Context:** Analyzing npm outdated output for major version bumps

**Learning:** Astro 6, @astrojs/cloudflare 13, and @astrojs/react 5 must upgrade together — they're tightly coupled. The cloudflare adapter v13 aligns with Astro 6 runtime changes and could affect D1/R2 binding access patterns. This is an all-or-nothing upgrade that warrants its own dedicated phase.
