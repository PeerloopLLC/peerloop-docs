# Conv Learnings - 2026-04-13

## 1. Stale Vite Cache After Major npm install
**Topics:** vite, astro, react, dev-environment

**Context:** After `npm install` that changed 200+ packages (including Cloudflare adapter 12→13), dev server threw "Invalid hook call" React errors.

**Learning:** Vite's pre-bundled dependency cache in `node_modules/.vite` can retain stale React bundles after major package changes, causing duplicate React instances. Always `rm -rf node_modules/.vite` after large dependency upgrades.

---

## 2. httpOnly Cookies Cannot Be Cleared Client-Side
**Topics:** auth, cookies, security

**Context:** Attempted to simulate session expiry by deleting auth cookies via `document.cookie` in browser console.

**Learning:** `httpOnly` cookies are invisible to JavaScript by design — `document.cookie` deletion has no effect. Must use server-side logout endpoint to clear them. This is the intended security behavior preventing XSS token theft.

---

## 3. Open Messaging Simplification Magnitude
**Topics:** messaging, architecture, simplification

**Context:** Removed relationship-based messaging gates in favor of open member-to-member messaging.

**Learning:** The relationship-based model required 3 SQL queries with 6 UNION/EXISTS subclauses. Open messaging reduced this to 1 simple IN query. The `messageableContactsSQL` went from 6 EXISTS clauses to a single `u.deleted_at IS NULL`. Net code reduction: 125 lines removed. Function signatures and API contracts unchanged — only internal rules simplified.
