# Conv Decisions - 2026-04-03

## 1. Add ConfirmModal to Certify Button
**Type:** Implementation
**Topics:** ux, certification, confirmation

**Trigger:** STUMBLE-AUDIT walkthrough found one-click teacher certification with no confirmation

**Options Considered:**
1. Leave as-is (fast, but inconsistent with Revoke/Cancel patterns)
2. Add ConfirmModal with default variant ← Chosen
3. Add ConfirmModal with danger variant (too alarming for positive action)

**Decision:** Added ConfirmModal with default (non-danger) variant to Certify button

**Rationale:** Consistency with existing ConfirmModal usage on Revoke (danger) and Cancel Session (danger). Certify is a positive but significant action — grants teaching privileges and earning capability.

**Consequences:** `CourseEditor.tsx` modified. Certify now shows "Certify as Teacher" dialog explaining consequences before proceeding.
