# Conv Learnings - 2026-06-20

## 1. Bare-numeric `rounded-N` utilities are no-ops in this Tailwind v4 setup
**Topics:** astro, design-tokens, spacing

**Context:** RG-COMMS grep found `rounded-8` (RecommendedCommunities) + `rounded-12` (FeedActivityCard, OnboardingNudgeBanner). Conv 309 had already flagged `rounded-8`=0px on Home.

**Learning:** `--radius-0..24/full` are defined in `tokens-primitives.css` `:root` but are NOT mirrored into any `@theme` block, so Tailwind v4 never generates `rounded-8`/`rounded-12` utilities → they resolve to nothing (0px, sharp corners). Named defaults (`rounded`, `rounded-lg`, `rounded-xl`, `rounded-full`) DO resolve. This is widespread — a repo-wide `rg 'rounded-[0-9]'` will find more; routed to `[SWEEP-SPACING-GREP]` #27.

**Pattern:** Conformant fix is the arbitrary form `rounded-[Npx]` (matches existing Matt components like CommunityCatalogCard) — not `rounded-N`.

## 2. Chrome MCP bridge transport breaks after a Claude Code re-login
**Topics:** cc-workflow, plato

**Context:** Live DOM-truth verify for the RG-COMMS slice was blocked; the user had re-logged-in Claude Code mid-session.

**Learning:** After a CC re-login the Chrome MCP *transport* can go dead: `tabs_context_mcp` returns "extension not connected" even though the `/chrome` slash-command reports connected. A full Chrome quit+reopen does NOT fix it (rules out Chrome — the break is the MCP transport, not the browser). Suspected-but-unverified fix: restart Claude Code (restart the VS Code terminal). Don't loop on retries / Chrome restarts — fall back to commit-and-defer or a Playwright DOM check (own chromium, no bridge). Captured in `memory/reference_chrome_bridge_island_stale_cache.md`.

## 3. Legacy `dark:` variants are a reliable "never-migrated" tell
**Topics:** matt-migration, css

**Context:** Distinguishing raw-legacy tab bodies from already-conformed islands during the RG-COMMS assessment.

**Learning:** Matt's design system is light-only. Any component carrying `dark:` variants (or `secondary-*`/`slate-*`/`indigo-*` tokens) was never migrated — a fast triage signal separating full-retrofit work from light conformance polish.
