# Conv Decisions - 2026-06-19

## 1. Standardize /profile spacing on bare Matt numerics + off-set normalization
**Type:** Strategic
**Topics:** matt-design, spacing, route-sweep

**Trigger:** StripeConnect shipped with arbitrary `[Npx]` + collapse-only; NotificationSettings used bare numerics + normalized off-set — divergence within the same route group.

**Options Considered:**
1. Bare Matt numerics + normalize off-set (match NotificationSettings; retro-fix StripeConnect) ← Chosen
2. Arbitrary `[Npx]` + collapse-only (match StripeConnect; NotificationSettings the outlier)
3. Bare numerics, collapse-only (hybrid)

**Decision:** Bare Matt numerics + off-set normalized; StripeConnect retro-fixed to match.

**Rationale:** Consistency with the first-swept tab; bare numerics read as the Matt scale; off-set normalization removes legacy Tailwind-scale classes entirely.

**Consequences:** StripeConnect re-opened + retro-fixed (commit `7e0fd706`); convention recorded in ledger + README for all future /profile + route-sweep work.
