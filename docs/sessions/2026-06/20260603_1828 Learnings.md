# Conv Learnings - 2026-06-03

## 1. conv-tasks.md no-shrink backstop catches /r-end under-capture
**Topics:** cc-workflow, dual-repo

**Context:** Conv 237 ended persisting only 5 follow-ups; the 37-task Conv-235 backlog was dropped from RESUME-STATE but survived on disk in conv-tasks.md. A multi-machine `/r-start` collision had orphaned the 37, and Conv 237's `/r-end` under-captured.

**Learning:** The disk-resident `conv-tasks.md` is the recovery source when a `/r-end` under-captures the carry-forward. The `/r-start` Step 7.5 no-shrink halt (`oldRows − #DONE ≠ TaskList`) is exactly the tripwire for this. When it fires, reconcile against the prior conv's committed session records + PLAN.md, not just RESUME-STATE — RESUME-STATE is the lossy surface.

## 2. A task can encode a wrong premise; read both sides before building
**Topics:** routing, cc-workflow

**Context:** [FEED-DETAIL] was written as "port /old/feed/* → /feed/[slug], a 4th [...tab] family" — but legacy `/feed` is a single 40-line SmartFeed page (`/old/feed.astro`); there is no slug family and nothing links to `/feed/[slug]`.

**Learning:** Carry-forward task descriptions are hypotheses, not facts. Reading the actual legacy source plus the live link graph falsified the multi-tab premise before any wrong code was written. The triad ([...tab].astro + _*-tabs.ts + SubNav) pattern is for multi-tab entity-detail; a personal smart feed is not that.

## 3. prov:sweep is outside the 5-gate baseline
**Topics:** testing, cc-workflow

**Context:** Running `prov:sweep` for the new sparkles icon surfaced 6 PRE-EXISTING errors (verified.svg missing entry, 3 untracked Astro stamps, 1 CourseHeader NODE-MISMATCH) that recent "5 gates green" claims (Convs 233/235/237) never caught.

**Learning:** The 5-gate baseline (tsc / astro check / lint / test / build) does NOT include `prov:sweep`, so provenance regressions slip past "gates green" claims silently. Tracked as [PROV-SWEEP-DEBT] #43; consider folding `prov:sweep` into `/w-codecheck` awareness so the prov axis is no longer invisible.
