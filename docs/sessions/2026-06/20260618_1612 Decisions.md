## 2. @matt-source conformance policy ‚Äî hybrid (tokenize-where-equivalent, keep token-less)
**Type:** Strategic
**Topics:** typography, matt-design, cc-workflow

**Trigger:** HOME-VERIFY found `@matt-source` primitives (1:1 Figma ports) carrying raw `text-[Npx]`; the conformance gate never said whether it reaches into them.

**Options Considered:**
1. Exempt @matt-source from the Type-token gate
2. Hybrid ‚Äî tokenize where a role token is exactly equivalent (zero visual change), keep genuinely token-less specs as recorded exceptions ‚Üê Chosen
3. Full tokenize (invent tokens even where none exists)

**Decision:** Hybrid. Shared @matt-source primitives get ledger rows + migrate once; verify per-route on each sweep. Small line-height *corrections* toward the ¬ß09 spec count as conformance, not regressions.

**Rationale:** Tokens were derived FROM the Figma specs, so exact-equivalent swaps are zero-change; genuinely token-less values (tracking, emoji size, #000) stay as sanctioned exceptions. Governs the rest of the route sweep.

**Consequences:** Recorded in `plan/typo-fdn/migration-ledger.md` ¬ß @matt-source policy + cross-reffed in the sweep README. 3 sanctioned exceptions logged.
