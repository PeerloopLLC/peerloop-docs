# Conv Learnings - 2026-06-16

## 1. Verify a flagged "legacy token" is actually a violation before acting
**Topics:** route-sweep, tailwind, design-tokens

**Context:** An Explore agent flagged `rounded-8`/`rounded-12` as a legacy bug ("rounded-12 = 48px").

**Learning:** This is Tailwind **v4** with `--radius-8: 8px`/`--radius-12: 12px` in `@theme`, so `rounded-8`/`rounded-12` are token-backed and render identically to `rounded-[8px]`/`rounded-[12px]` — the bare form is arguably the *purer* token usage. The v3 "scale × 4px" assumption was wrong. Always confirm the theme/token backing before treating a bare-scale utility as a survivor.

**Pattern:**
`grep -nE "@theme|--radius-N" src/styles/tokens-primitives.css` to confirm token backing; count instances before declaring a Rule-of-Three fix.

## 2. Distinguish HMR-transient console errors from real ones
**Topics:** browser-verify, vite, dom-truth

**Context:** After a 3-edit sequence on CoursesFilters, the console showed `SegmentedPills is not defined`.

**Learning:** The errors carried *different* HMR cache-bust tokens (`?t=…`) — they were intermediate hot-reload builds (import removed before JSX replaced), not the live build. A hard reload + re-check showed zero errors. Clear the console and hard-reload before trusting load-time console errors after a multi-edit HMR sequence.

## 3. DOM truth for the clearable-select fix
**Topics:** dom-truth, forms, browser-verify

**Context:** Verifying the Select `clearable` fix.

**Learning:** Asserted `option.disabled === false` directly via `javascript_tool` rather than screenshot — the duration select (value="short", empty option `disabled:false`) was the definitive proof that a value-selected dropdown can return to "All". Screenshots can't show option-disabled state.

## 4. Rule-of-Three extended to cross-cutting token concerns
**Topics:** route-sweep, SOP

**Context:** User reframed Step-7 cross-cutting handling.

**Learning:** The Rule-of-Three (extract a primitive at 3 instances) generalizes to token/styling cleanups: ≥3 distinct sites ⇒ fix now (consolidate every site, at whatever route it ripened on); <3 ⇒ watch. **Either way, log it** — that's the SOP.
