## 2. System feed (townhall) reaction/comment participation = admin-only
**Type:** Implementation
**Topics:** stream, auth

**Trigger:** Needed a participation rule for the townhall reaction/comment endpoints; the source townhall feed (GET/POST) is admin-only since SYS-RENAME (Conv 259).

**Options Considered:**
1. Treat the-commons membership like a community
2. Admin-only, consistent with the source townhall feed ← Chosen

**Decision:** `canParticipate({type:'system'})` returns true only for admins.

**Rationale:** Members never reach the System feed directly (they get its content via Announcements, deferred); gating reactions/comments to admins matches the feed's existing lockdown.

**Consequences:** Townhall reaction/comment tests now seed an admin actor + a non-admin 403 case.
