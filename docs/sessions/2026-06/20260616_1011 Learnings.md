# Conv Learnings - 2026-06-16

## 1. `bg-primary` is transparent; the Matt brand-blue background token is `bg-text-primary`
**Topics:** astro, matt-design, tokens

**Context:** Migrated SmartFeed's active filter-tab pill from legacy `bg-primary-600` to a guessed `bg-primary` — the tab went invisible (white text on nothing).

**Learning:** `bg-primary` is NOT a defined utility in this project; it resolves to `rgba(0,0,0,0)` (transparent). The Matt brand-blue **background** token is `bg-text-primary` (= `#0777b6` = `rgb(7,119,182)`, mirroring `text-text-primary`). `bg-primary-light` (= `#f1f9ff`) is the light brand surface. Proven-valid Matt palette (via OnboardingNudgeBanner usage): `text-text-{default,tertiary,primary}`, `bg-primary-light`, `bg-text-primary`, `border-border-default`, `rounded-{8,12}`, `text-body-*`.

**Pattern:**
`getComputedStyle`-probe an unfamiliar token before shipping it:
```js
const d = document.createElement('div');
d.className = 'bg-x';
document.body.appendChild(d);
getComputedStyle(d).backgroundColor
```

## 2. Gates (tsc/astro/lint) don't catch CSS errors — browser/DOM verify is mandatory for visual work
**Topics:** testing, matt-design, plato

**Context:** The invisible filter tab passed tsc, astro check (0/0/0), and lint clean — only the browser caught it.

**Learning:** A bad Tailwind class is a no-op the type system can't see. For any visual/token change, the gate is necessary but not sufficient; render it and DOM-verify (`getComputedStyle`/`getBoundingClientRect`). Reinforces `feedback_dom_truth_over_screenshots`.

**Topics:** matt-design

**Context:** Assumed OnboardingNudgeBanner (Conv 054) was a pre-Matt straggler; reading it showed full Matt tokens + MattIcon + Button.

## 5. TodoWrite has no parent/child — model "groups as tasks, pages as sub-tasks" as task + README checklist
**Topics:** cc-workflow

**Context:** User wanted route groups as tasks with per-page sub-tasks.

**Learning:** TaskCreate supports only blocks/blockedBy, no hierarchy. The durable per-page checklist lives in `plan/route-migration/README.md` (☐/☑ rows); TodoWrite holds the 14 group tasks. The README is the conv-surviving "checklist of done routes."
