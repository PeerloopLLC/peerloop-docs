## 6. FeedActivityCard source tint — keep the 3-way distinction (course→info)
**Type:** Strategic
**Topics:** design-system, palette

**Trigger:** PALETTE-FDN maps primary→brand AND indigo→brand, which would collapse the feed's system vs course colour-coding to one tint.

**Options Considered:**
1. Accept the collapse (both brand)
2. Keep 3-way: map course→info (blue) ← Chosen
3. Drop bg tints entirely

**Decision:** system=brand, community=neutral, **course=info** — preserve the source distinction using the role palette.

**Rationale:** Source colour-coding is the component's explicit purpose; collapsing it loses real signal. A single semantic divergence from the literal indigo→brand is worth it.

**Consequences:** 3-way source tint applied in FeedActivityCard (`02ba8664`).
