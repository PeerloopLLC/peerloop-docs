# Conv Decisions - 2026-06-07

## 1. Keep selectable empty-value options as real options, not placeholder
**Type:** Implementation
**Topics:** matt-design, primitives, forms

**Trigger:** TownHallFeed's two selects each have a value="" first option that is a real, selectable choice.

**Options Considered:**
1. Use the Select primitive's `placeholder` prop for the first option
2. Keep the first option as a normal `options` array entry ← Chosen

**Decision:** Pass "None (general post)" / "All posts" as ordinary `options` entries with value="".

**Rationale:** The primitive's `placeholder` renders a *disabled* option; these choices must stay selectable (clearing a course tag / showing all posts).

**Consequences:** Both selects behave identically to the pre-swap raw selects; only styling changed.
