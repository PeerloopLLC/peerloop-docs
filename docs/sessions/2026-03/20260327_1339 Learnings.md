## 2. BBB Analytics JWT Secret Source Chain
**Topics:** bbb, security, documentation

**Context:** User asked where the HS512/BBB_SECRET claim in the email draft came from.

**Learning:** The claim traces through 3 layers: (1) Blindside Networks PDF attachment on CD-038 specifies HS512 JWT auth, (2) bigbluebutton.md documents the secret as the same BBB shared secret, (3) bbb-analytics.ts implements verification using env.BBB_SECRET. The email confirmation question is a sanity check — if wrong, all analytics callbacks silently fail auth (401). Good practice to confirm assumptions about shared secrets even when documentation seems clear.
