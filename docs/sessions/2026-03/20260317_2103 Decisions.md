# Conv Decisions - 2026-03-17

## 1. Improve reset-d1.js Circular Dependency Messaging
**Type:** Implementation
**Topics:** d1, testing

**Trigger:** Staging reset showed `⚠️ Circular dependency detected involving courses` — alarming wording for a benign self-referential FK.

**Options Considered:**
1. Leave as-is — it works, just noisy
2. Suppress the warning entirely for self-references ← Chosen (partially)
3. Distinguish self-references from true cycles with different severity levels ← Chosen

**Decision:** Modified `reset-d1.js` to detect self-references vs true cross-table cycles separately. Self-references show `ℹ️ Self-reference on {table} — safe to ignore`. True cycles show `⚠️ Circular dependency: {tableA} ↔ {tableB} — this may cause DROP failures and should be fixed in the schema`.

**Rationale:** The original code only knew the target table inside the `visiting.has()` guard. Moving detection into the ref loop where both tables are known allows precise classification. Self-references are a normal relational pattern; true cycles are a schema design issue worth flagging.

**Consequences:** `scripts/reset-d1.js` updated. Verified on staging reset — shows `ℹ️` for `courses` self-reference.

## 3. Dynamic Test Doc Discovery in sync-gaps.sh
**Type:** Implementation
**Topics:** docs-infra, cc-workflow

**Trigger:** sync-gaps.sh reported 75 undocumented test files — all false positives caused by only checking `TEST-COVERAGE.md` when test docs are split across 3+ files.

**Options Considered:**
1. Hardcode all 3 filenames (`TEST-COVERAGE.md`, `TEST-COMPONENTS.md`, `TEST-UNIT.md`)
2. Dynamically discover all `TEST-*.md` files via glob ← Chosen

**Decision:** Use `ls "$REF_DOCS"/TEST-*.md` to auto-discover all test doc files. Loop through each when checking whether a test file is documented.

**Rationale:** Hardcoding broke twice in this conv (first missed `TEST-COMPONENTS.md`, then missed `TEST-UNIT.md`). Dynamic discovery eliminates the coupling — adding `TEST-API.md` or any future sub-doc requires zero script changes.

**Consequences:** `sync-gaps.sh` updated. 75 false positives eliminated. 326/326 test files now correctly reported as documented.
