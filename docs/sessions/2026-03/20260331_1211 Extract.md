### New Patterns
- **findBy in extractPath:** `courses.findBy(title,$persona.courseTitle).id` — array element lookup by field match. Requires parseDotPath() for paren-aware splitting.

- **Course flattening:** runtimeOverrides.courseIndex triggers flattenCourseData() which copies courses[N].* onto persona top level.
- **Scenario composition:** Chain steps compose runs with per-invocation actor bindings and runtime overrides. Same run invoked multiple times with different bindings.
