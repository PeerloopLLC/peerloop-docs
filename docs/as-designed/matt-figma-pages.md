**Conv 183 Main Nav architecture finding (informs strict-B rebuild of Sidebar.tsx):** the "Main" Property 1 value is not "expand on click" — it's the visual mode where the **active parent + its currently-active child** are surfaced together inside one white pill, with sibling parents (Courses, Peer Teachers) rendered outside the pill at the Default state. Each nav row supports an optional second-line "Sub Text Label" beneath the main label. The Conv 175/176 Sidebar.tsx (220/70px collapse-toggle, 5 hardcoded flat items, no nesting) does NOT mirror this; a faithful rebuild requires 3 new components (MainNav container, NavItem with 3 Property 1 variants, NavSubItem with 2 Property 1 variants) and a data model for nav items with optional children + sub-text.
